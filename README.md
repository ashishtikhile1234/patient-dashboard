# patient-dashboard

> Coalition Technologies — FED Skills Test  
> Patient Dashboard built from Adobe XD design with live API integration.

## Features
- 📡 Live data from Coalition Technologies Patient Data API
- 📊 Blood pressure chart using Chart.js
- 🩺 Vital signs, diagnostic list, and lab results for **Jessica Taylor**
- 📱 Responsive layout
- ✅ Pixel-perfect match to Adobe XD design

## Tech Stack
- HTML5
- Vanilla CSS
- Vanilla JavaScript (ES6+)
- Chart.js v4

## API
- Endpoint: `https://fedskillstest.coalitiontechnologies.workers.dev`
- Auth: HTTP Basic Auth (credentials generated via `btoa()`)

## Run Locally
Just open `index.html` in your browser — no build step required.
